# tank war

## 介绍

这是一个基于 web canvas 开发的坦克大战游戏,使用了 `typescript` , `react` 和浏览器提供的
 `canvas` api 。
 
## 运行项目

需要先准备 node、npm 环境

在项目目录下输入以下下命令行来搭建环境

```bash
yarn
```

或者

```bash
npm install
```

然后输入以下命令行来运行

```bash
yarn serve
```

或者

```bash
npm run serve
```

打开浏览器 [开发页面](http://localhost:8083) 就成功启动项目了
